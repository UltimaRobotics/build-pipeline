About Custom Repositories
Adding a Git Repository

You can add custom package feeds from Git repositories to include packages that are not available in the official OpenWrt package feeds.

Authentication for Private Repositories

For private repositories, you need to provide an authentication token or key:

  • For GitHub: Create a personal access token with appropriate permissions.
  • For other Git hosts: Follow their authentication documentation.
Security Note: Authentication tokens are stored securely but should be treated with care. Use tokens with minimal permissions needed for read access to the repository.
